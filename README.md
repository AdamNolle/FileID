# 📂 FileID

### 🚧 Work in Progress
This app is currently in active development. Features are being added and refined frequently.

---

> **AI-Powered File Organization.**

**FileID** is a macOS application that uses AI to bring order to your digital life. It moves beyond simple "naming rules" to perform deep **Semantic Restructuring**—understanding what your files actually *are* and organizing them.


---

## ✨ Key Features

- **🧠 Semantic Clustering**: AI analyzes file content and metadata to group related items (e.g., "Tax Receipts," "Vacation 2023," "Legacy Code").
- **🧼 Cleanup View**: Identify duplicates and junk files in seconds.

## 🛠️ Technical Specifications

- **OS**: macOS (Native Swift/SwiftUI)
- **Engine**: On-device Vision and Natural Language processing.

## 🚀 Usage

1. **Scan**: Point FileID at your most chaotic folder.
2. **Draft**: Let the AI generate a "Master Plan" on the Infinite Canvas.
3. **Review**: Tweak the categories and verify the moves.
4. **Execute**: One-tap to transform your file structure into perfection.
